## 기능목록

### 입력 및 출력
- [ ] 자동차 이름을 콤마로 구분해 입력
- [ ] 시도 횟수를 입력
- [ ] 각 시도별 결과를 출력
- [ ] 최종 우승자 출력
### 도메인 기능
- [x] 자동차 이름 입력값을 자동차들로 변환해 저장
- [ ] 랜덤값을 통해 자동차 전진
- [ ] 전진 여부에 따라 자동차의 위치를 저장하며 업데이트
- [ ] 시도 횟수 입력값을 변환해 저장
- [ ] 시도 횟수 만큼 자동차들 전진시키기
- [ ] 자동차들 중 우승자 판단
- [x] 각 시도별 결과 출력을 위한 DTO 생성 (자동차 이름, 자동차 위치 저장)
- [ ] 최종 우승자 출력을 위한 DTO 생성
